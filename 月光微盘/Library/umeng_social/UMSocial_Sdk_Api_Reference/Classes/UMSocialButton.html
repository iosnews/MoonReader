Instance Methods

initWithButtonName:controllerService:controller:

按钮初始化方法

- (id)initWithButtonName:(NSString *)buttonName controllerService:(UMSocialControllerServiceComment *)controllerService controller:(UIViewController *)controller

Parameters

buttonName

按钮类型，可以指定UMSocialShare,UMSocialComment，UMSocialLike、UMSocialAccount分别代表分享、评论、喜欢、个人中心 或者UMShareToSina、UMShareToTencent等代表各个微博平台

controllerService

UMSocialControllerService对象，如果是评论或者喜欢，代表不同的评论、喜欢对象。可以传nil，使用默认数据对象

controller

分享编辑页面、评论页面等将弹到的UIViewController对象
