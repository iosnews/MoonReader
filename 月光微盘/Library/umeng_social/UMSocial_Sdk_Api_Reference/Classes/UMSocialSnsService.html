Parameters

url

传入的url

Return Value

wxApiDelegate 实现微信代理对象

Discussion

如果你要处理自己的url，你可以把这个方法的实现，复制到你的代码中：

if ([url.description hasPrefix:@“sina”]) {

return (BOOL)[[UMSocialSnsService sharedInstance] performSelector:@selector(handleSinaSsoOpenURL:) withObject:url];

} else if([url.description hasPrefix:@“wx”]){

return [WXApi handleOpenURL:url delegate:(id <WXApiDelegate>)[UMSocialSnsService sharedInstance]];

}

+ (void)presentSnsController:(UIViewController *)controller appKey:(NSString *)appKey shareText:(NSString *)shareText shareImage:(UIImage *)shareImage shareToSnsNames:(NSArray *)snsNames delegate:(id<UMSocialUIDelegate>)delegate

Parameters

controller

在该controller弹出分享列表的UIActionSheet

appKey

友盟appKey

shareText

分享编辑页面的内嵌文字

shareImage

分享内嵌图片,用户可以在编辑页面删除

snsNames

你要分享到的sns平台类型，该NSArray值是UMSocialSnsPlatformManager.h定义的平台名的字符串常量，有UMShareToSina，UMShareToTencent，UMShareToRenren，UMShareToDouban，UMShareToQzone，UMShareToEmail，UMShareToSms等

delegate

实现分享完成后的回调对象，如果不关注分享完成的状态，可以设为nil
