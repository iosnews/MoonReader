UMSocialData.m

Overview

一个UMSocialData对象标识一个分享资源，用一个identifier字符串作为标识，你可以为此对象设置分享内嵌文字、分享图片等，可以获取到对应的分享数、评论数。同时UMSocialData类方法用来设置appKey和打开log等全局设置。
