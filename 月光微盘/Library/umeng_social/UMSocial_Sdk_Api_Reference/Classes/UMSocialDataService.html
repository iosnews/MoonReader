Inherits from NSObject

UMSocialDataService.m

Overview

底层数据接口对象，用一个UMSocialData来初始化，此对象的方法有在直接发送微博、发送评论等。可以通过socialData属性来获取分享数、评论数，设置分享内嵌文字等。
